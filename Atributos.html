Atributos HTML

No.

Atributos

Definición

1

id="texto"

Establece un indicador único a cada elemento

2

class="texto"

Establece la clase CSS que se aplica a los estilos del elemento

3

style="texto"

Aplica de forma directa los estilos CSS de un elemento

4

title="texto"

Establece el título del elemento (Mejora la accesibilidad)

5

controls

Indica si el navegador debe mostrar los controles multimedia del reproductor integrado al usuario.

6

align

Este atributo sigue funcionando pero no es aconsejable usarlo ya que está en desuso. Lo recomendable es usar reglas CSS en su lugar.

7

download

Si este atributo se añade al elemento HTML, el recurso indicado podrá descargarse al dispositivo del usuario.

8

async

Indica que el script debe ejecutarse de forma asíncrona.

9

bgcolor

Te permite añadir un color de fondo al elemento HTML. El color debe ser añadido en el modo de color RGB, por ejemplo #FF00AA. Este atributo sigue funcionando pero no es aconsejable usarlo ya que está en desuso. Lo recomendable es usar reglas CSS en su lugar.

10

checked

Usado para definir que un elemento HTML debe estar marcado por defecto al cargar la página web.
